## Variant Types

The training database classifies words into 4 types:

1. **single** (85-90%): Only one stress pattern
2. **morphological_variant** (10%): Different POS/features → different stress
3. **grammatical_homonym** (5%): Same morphology but different meanings → different stress
4. **free_variant** (<1%): Multiple acceptable stresses, same meaning

## Training Strategy

### Weighted Sampling

Due to class imbalance, use weighted sampling:

```python
import sqlite3

conn = sqlite3.connect('../data/stress_training.db')

# Oversample rare classes
weights = {
    'single': 1.0,
    'morphological_variant': 8.0,
    'grammatical_homonym': 15.0,
    'free_variant': 20.0
}

# Build training set with resampling
train_data = []
for variant_type, weight in weights.items():
    rows = conn.execute("""
        SELECT * FROM training_entries
        WHERE split='train' AND variant_type=?
    """, (variant_type,)).fetchall()

    # Replicate based on weight
    train_data.extend(rows * int(weight))
```

### Features for Model

Extract features from `training_entries`:

```python
def extract_features(row):
    import json

    lemma = row['lemma']
    pos = row['pos']
    features_json = json.loads(row['features_json'] or '{}')
    definition = row['definition'] or ""

    # Character-level features
    char_features = [
        len(lemma),
        lemma.count('а'), lemma.count('е'), lemma.count('и'),  # vowels
        # ... more features
    ]

    # Morphological features
    morph_features = [
        features_json.get('Case', 'Nom'),
        features_json.get('Gender', 'Masc'),
        features_json.get('Number', 'Sing'),
    ]

    # Context features
    has_definition = 1 if definition else 0

    return char_features + morph_features + [has_definition]
```

### Example Training Loop (LightGBM)

```python
import lightgbm as lgb
import sqlite3
import json

# Load data
conn = sqlite3.connect('../data/stress_training.db')
rows = conn.execute("""
    SELECT lemma, stress_indices, pos, features_json, definition
    FROM training_entries
    WHERE split='train'
""").fetchall()

# Extract features and labels
X = [extract_features(row) for row in rows]
y = [json.loads(row['stress_indices'])[0] for row in rows]  # First stress index

# Train
model = lgb.LGBMClassifier(
    num_leaves=31,
    max_depth=-1,
    learning_rate=0.1,
    n_estimators=100
)
model.fit(X, y)

# Save
import pickle
with open('../models/stress_model_v1.pkl', 'wb') as f:
    pickle.dump(model, f)
```

## Evaluation Metrics

- **Accuracy**: % of correct stress predictions
- **Per-variant accuracy**: Accuracy broken down by variant type
- **Confusion matrix**: Which stress positions are commonly confused
- **Character-level analysis**: Accuracy by word length, vowel count, etc.

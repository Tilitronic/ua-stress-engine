# Stress Prediction Data

Output databases and datasets for stress prediction training.

## Files

- `stress_training.db` - SQLite training database (generated by export_training_db.py)
- `.gitignore` - Ignore generated database files

## Database: stress_training.db

**Source**: Generated from `src/data_management/export/export_training_db.py`

**Schema**:

```sql
-- Main training data
training_entries (
    id, lemma, form, stress_indices, pos, features_json,
    definition, variant_type, is_disambiguable,
    split, frequency, source_tables
)

-- Metadata
db_metadata (key, value)
db_stats (metric, value)
```

**Size**: ~50-100MB (depends on merged DB size)

**Regeneration**:

```bash
python -m src.data_management.export.export_training_db --overwrite
```

## Variant Types

- **single**: Only one stress pattern (majority of words)
- **free_variant**: Multiple acceptable stresses, same meaning (e.g., пОмилка/помИлка)
- **grammatical_homonym**: Different meanings with different stress (e.g., замОк/зАмок)
- **morphological_variant**: Different POS/features with different stress

## Statistics

Check database statistics:

```python

conn = sqlite3.connect('src/stress_prediction/data/stress_training.db')
cursor = conn.cursor()

cursor.execute("SELECT * FROM db_stats")

    print(f"{metric}: {value}")
```
